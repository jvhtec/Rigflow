# RigFlow Documentation Hub / Centro de Documentación

## EN
RigFlow provides an AutoLISP command (`RIGFLOW`) and required audio rigging blocks for fast, repeatable insertion workflows in AutoCAD.

### Included files
- `RIGFLOW_AUDIO_V3_5_6_PRELOADED_BLOCKS.lsp`
- `1T AUDIO.dwg`
- `2T AUDIO.dwg`

### Documentation map
- Architecture: [`architecture.md`](./architecture.md)
- Installation Manual: [`installation manual.md`](./installation%20manual.md)
- User Manual: [`user manual.md`](./user%20manual.md)
- Persistence Guide: [`persistence-and-drawing-data.md`](./persistence-and-drawing-data.md)
- Agent/Contributor Guide: [`agents.md`](./agents.md)

### Non-negotiable operational warning
> Ensure required block definitions are available (`1T AUDIO`, `2T AUDIO`) before running production layouts.
>
> Best practice: embed these definitions in your project template by inserting each block at least once and saving the template.
